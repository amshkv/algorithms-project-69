# nodejs-package

## Requirements

* Unix system (Macos/Linux)
* Nodejs 22.x+
* Make

## Setup

```bash
make install
```

## Run tests

```bash
make test
```

See other commands in [Makefile](./Makefile)

## Run tests with coverage

See [sonar-project.properties](./sonar-project.properties#1:2):

```bash
make test-coverage
# see ./coverage
```
